Getting Started.

About The Engine.

Violet Engine is a fnf engine coded from the ground up for the main purpose of improving the modding experience.

The engine has FULL soft modding support made easy with our own Modding API.


All the editors' UIs in the engine is powered by our very own UI Library called LemonUI, which you can get here on GitHub.

Getting The Engine.

You can get the engine on github from here GitHub.

Scripting.

General Information

The engine has support for Lua, Basic Psych Lua, HScript, Basic HXC, and Python.
HXC Scripts can go anywhere in your mod's folder for them to run, just like V-Slice.

All script calls have on and upon variants. (example: create -> onCreate -> uponCreate)

Song Scripting

Song scripts go in the following folders:
• mods/YourModFolder/songs/
• mods/YourModFolder/songs/yourSong/scripts/
• mods/YourModFolder/data/scripts/songs/
